This repository contains raw data in output from the simulations. 

Averages files include simulator outputs averaged per heart beat over 60seconds simulation.
Waveforms files include continuos simulator outputs over 60seconds simulation.

PEEP = positive end-expiratory pressure 

PCS = post cardiac surgery patients 
TV = tidal volume test 

PCV = pressure control ventilation 
VCV = volume control ventilation
IE = Inspiratory:Expiratory ratio
